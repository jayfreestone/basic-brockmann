The
Basic Brockmann

for web designers

A set of basic grid mixins written in SASS, for use with Libsass, owing to Steve Hickey.

Setup

Include _grid.scss & _mixins.scss in your project, then declare these variables in a preceeding file:

$max-width: 1360px;  
$column-width: 6.38298%;  
$gutter-width: 2.12766%;  
$maximum-columns: 12;

Usage

Define the main container with the %container placeholder:

.container {  
  @extend %container;  
}

Then define the widths of elements with the col mixin:

main {  
  @include col(8);  
}

By default col will assume the parent element contains the maxium number of columns specificed in $maximum-columns — in this example 12. If you are nesting elements and wish to instead declare how many columns the parent has you may pass in a second number:

aside {  
  @include col(2, 6);  
}

This translates to: ‘span two of six columns’.

The final column in a row must be declared by passing in the omega flag, or optionally using the omega mixin:


aside {  
  @include col(2, 6, omega);  
}

// Or:

aside {  
  @include col(2, 6);  
  @include omega;  
}

The first is prefferable as it avoids outputting CSS that will be rewritten.

Currently when passing in the omega flag the parent needs to be declared, ensuing omega is the last option passed into the mixin. This would, for instance, not be valid: @include col(2, omega).
